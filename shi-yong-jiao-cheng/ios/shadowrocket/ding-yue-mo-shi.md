# 订阅模式

* `系统要求：iOS 9 及以上`
* `软件版本：Shadowrocket 最新版`
* `设备要求：iPhone / iPad`



{% hint style="danger" %}
此应用已在国区下架，请自行准备非国区 Apple ID 下载。
{% endhint %}

将 Shadowrocket 下载到本地，登入 dlercloud.com 用户中心，找到 ALL-IN-ONE 模块，导入自己喜爱的模式即可
